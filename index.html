Часть 1. Почему Python?

Python — интерпретируемый, высокоуровневый язык с динамической типизацией. Его философия делает код понятным и выразительным, а экосистема позволяет решать задачи от веб‑разработки до искусственного интеллекта.

  • ✅ Читаемость и лаконичность
  • ✅ Огромное сообщество и библиотеки
  • ✅ Подходит для первого языка программирования

Структура простой программы

# переменные и типы
name = "Алексей"
age = 17
height = 1.75
is_student = True

# условный оператор
if age >= 18:
    print("Доступ разрешён")
else:
    print("Доступ ограничен")

# цикл for
for i in range(5):
    print(f"Квадрат числа {i} = {i*i}")
💡 Обратите внимание: блоки кода выделяются отступами (пробелами). Это не просто стиль — это синтаксическое правило.

Часть 2. Быстрый старт

Установка и запуск

  • Скачайте Python с официального сайта python.org (версия 3.10 или новее).
  • Во время установки обязательно отметьте галочку «Add Python to PATH».
  • Проверьте установку в терминале/командной строке: python --version.
  • Для написания кода мы будем использовать PyCharm Community — это профессиональная среда разработки, созданная специально для Python. (Альтернативы: VS Code, IDLE, но на курсе акцент на PyCharm).

Как выполнить код

  • Интерактивный режим (REPL): введите python в терминале и пишите команды построчно.
  • Запуск скрипта: сохраните код в файл с расширением .py (например, my_program.py) и выполните python my_program.py.
  • В PyCharm: создайте новый проект, добавьте файл .py и нажмите зелёную кнопку запуска.
🔁 Совет: для комфортной разработки используйте виртуальное окружение (python -m venv venv), чтобы изолировать проекты друг от друга.

2 Простой калькулятор

Создайте программу, которая запрашивает у пользователя два числа и операцию (+, -, *, /). Выполняет вычисление и выводит результат. Программа должна обрабатывать деление на ноль и неверный ввод (например, вместо числа введена буква).

# Решение задания 2
while True:
    try:
        a = float(input("Введите первое число: "))
        op = input("Введите операцию (+, -, *, /): ")
        b = float(input("Введите второе число: "))
        if op == '+':
            result = a + b
        elif op == '-':
            result = a - b
        elif op == '*':
            result = a * b
        elif op == '/':
            if b == 0:
                print("Ошибка: деление на ноль!")
                continue
            result = a / b
        else:
            print("Ошибка: неизвестная операция")
            continue
        print(f"{a} {op} {b} = {result}")
        break
    except ValueError:
        print("Ошибка: введите числа, а не текст")

3 Геометрия с модулем math

Напишите программу, которая запрашивает радиус круга (положительное число) и вычисляет его площадь и длину окружности, используя значение π из модуля math. Выведите результаты с точностью до двух знаков после запятой.

# Решение задания 3
import math

while True:
    try:
        r = float(input("Введите радиус круга (положительное число): "))
        if r <= 0:
            print("Ошибка: радиус должен быть больше нуля")
            continue
        area = math.pi * r ** 2
        circumference = 2 * math.pi * r
        print(f"Площадь круга: {area:.2f}")
        print(f"Длина окружности: {circumference:.2f}")
        break
    except ValueError:
        print("Ошибка: введите число")
