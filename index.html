2 Простой калькулятор

Запросите два числа и операцию (+, -, *, /). Выведите результат. Обработайте деление на ноль и нечисловой ввод.

📘 Решение
while True:
    try:
        a = float(input("Первое число: "))
        op = input("Операция (+, -, *, /): ")
        b = float(input("Второе число: "))
        if op == '+': result = a + b
        elif op == '-': result = a - b
        elif op == '*': result = a * b
        elif op == '/':
            if b == 0:
                print("Ошибка: деление на ноль!")
                continue
            result = a / b
        else:
            print("Неизвестная операция")
            continue
        print(f"{a} {op} {b} = {result}")
        break
    except ValueError:
        print("Ошибка: введите числа")

3 Геометрия с модулем math

Запросите радиус круга (положительное число). Вычислите площадь и длину окружности, используя math.pi. Результат округлите до двух знаков.

📘 Решение
import math
while True:
    try:
        r = float(input("Радиус круга: "))
        if r <= 0:
            print("Радиус должен быть > 0")
            continue
        area = math.pi * r ** 2
        circ = 2 * math.pi * r
        print(f"Площадь: {area:.2f}\nДлина окружности: {circ:.2f}")
        break
    except ValueError:
        print("Ошибка: введите число")

4 Числа Армстронга ★

Число Армстронга (трёхзначное) равно сумме кубов его цифр. Например, 153 = 1³ + 5³ + 3³. Программа должна найти и вывести все трёхзначные числа Армстронга (от 100 до 999).

📘 Решение
for num in range(100, 1000):
    a = num // 100          # сотни
    b = (num // 10) % 10    # десятки
    c = num % 10            # единицы
    if a**3 + b**3 + c**3 == num:
        print(num)
